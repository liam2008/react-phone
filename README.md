phone-react

所要实现的效果

单纯的练习项目，通过boostrap作为css样式库，做出不同react组件，数据存储到localstorage里面，实现一个登陆系统的增删改查。

安装

    	> git clone http://192.168.0.203:31590/liam/phone-react.git
    	> cd phone-react
    	> npm install
    	> npm start

使用

    	> npm install
    	> npm start

用到的技术

- Bootstrap
- React
- React-router
